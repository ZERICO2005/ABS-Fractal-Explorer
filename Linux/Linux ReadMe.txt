ABS-Fractal-Explorer
v1.0.1 Linux | 2023/08/15 | zerico2005

The Linux version of ABS-Fractal-Explorer was compiled and tested on Ubuntu.
I am not experinced with Linux, so I don't know if it will run on other distributions.
One thing that can help is ensuring your GPU/Graphics drivers are up to date.

You might need to get the following for the program to run: (You could probably get them in the terminal)
- SDL2
- gcc/g++
- mingw
- pthread
