v1.0.2 Linux | 2023/08/17 | zerico2005

The Linux version of ABS-Fractal-Explorer was compiled and tested on Ubuntu. You may need to manually install some libraries for it to run on your system.
One thing that can help is ensuring your GPU/Graphics drivers are up to date.

Needed Libraries:

libOpenCL.so.1
libSDL2-2.0.so.0
libm.so.6
limc.so.6
ld-linux-x86-64.so.2

You could try looking up and running sudo apt install commands for them
